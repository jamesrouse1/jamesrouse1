### About Me

I work in the Bioinformatics Shared Resource at Cold Spring Harbor Laboratory, where I develop tools and workflows for RNA-seq, ATAC-seq, ChIP-seq, single-cell analysis, visualization, and reproducible project delivery.

My work focuses on making complex genomics analyses easier to run, easier to inspect, and easier to reproduce.

### Featured Projects
